## Functions

```
01. Supports tab manager
02. To view a photo of the site, you need to copy the link and press Shift + V 
03. User-Agent changer
04. Fake accept language
05. Disable or enable JavaScript
06. Disable or enable Anti Phishing
07. Disable or enable WebRTC
08. MyIP information
09. Download website as HTML
10. Download website as PDF
11. Open file for preview
12. Copy website link
13. Open google translate
14. Disable or enable proxy server (vpn)
15. SSL certificate information
16. Search engine changer
```
